## 🏗️ Over Euramax

**Euramax is al meer dan 35 jaar dé specialist in hoogwaardige bouwmaterialen en gevelbekleding** voor mobility, building en architectural klanten. Wij zijn toegewijd aan het leveren van **cladding materialen die voldoen aan de hoogste standaarden** en een leven lang meegaan.

### 🎯 Onze Expertise

- **🏠 Dakbedekkingssystemen**: Van traditionele pannen tot moderne membranen

- **📋 Maatwerk Advies**: Persoonlijke begeleiding voor elk project

## 🚀 Snelle Start

### ⚡ Website Lokaal Bekijken
```bash
# Clone de repository
git clone https://github.com/infopinkbalaclava-design/Euramax.git
cd Euramax

# Start lokale webserver
python3 -m http.server 8080 --directory static

# Open website in browser
# Ga naar: http://localhost:8080/
```
### 🌐 Browser Compatibiliteit
De website is getest en werkt optimaal op:
- ✅ Chrome 90+ 
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+
- ✅ Mobile browsers (iOS Safari, Chrome Mobile)

### 📋 Vereisten

- Python 3.11+ (voor lokale ontwikkeling)
- Moderne webbrowser (Chrome, Firefox, Safari, Edge)
- Internetverbinding voor fonts en externe assets
